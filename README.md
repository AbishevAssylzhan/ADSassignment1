## Tasks

### Task 1 – Print Digits
This function prints each digit of a number one by one.

### Task 2 – Average of Elements
The program finds the sum of array elements recursively and calculates the average.

### Task 3 – Prime Number Check
Checks if a number is prime using recursion.

### Task 4 – Factorial
Calculates the factorial of a number.

### Task 5 – Fibonacci
Finds the n-th Fibonacci number.

### Task 6 – Power Function
Calculates a number raised to a power.

### Task 7 – Reverse Output
Prints array elements in reverse order.

### Task 8 – Check Digits in String
Checks if the string contains only digits.

### Task 9 – Count Characters
Counts the number of characters in a string.

### Task 10 – GCD
Finds the greatest common divisor using recursion.
